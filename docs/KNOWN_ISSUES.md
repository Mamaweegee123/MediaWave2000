# Known Issues: v0.1.0 Private Beta

- Windows and macOS packages are currently unsigned. Windows SmartScreen and
  macOS Gatekeeper may display warnings.
- NetTV depends on third-party websites and may stop working when those sites
  change behavior.
- NetTV cookies and authentication are local tester configuration and are never
  included in beta packages.
- Very large media libraries may take time to scan and generate thumbnails on
  first use.
- Some unusual media codecs may require conversion before playback.
- Display scaling, custom themes, external drives, network shares, and unusual
  font configurations need broader beta coverage.
- The beta has no automatic updater. Replace the application manually when a
  newer private-beta package is provided.

Report the operating system, display scaling, selected theme, reproduction
steps, and relevant screenshots when submitting an issue. Do not include
personal media, credentials, or private paths unless explicitly requested.
