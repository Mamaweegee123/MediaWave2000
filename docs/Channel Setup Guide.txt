===============================================================================
CHANNEL SETUP GUIDE
Build your own fake cable lineup. No coax required.
===============================================================================

CHANNEL FOLDER STRUCTURE
-------------------------
MediaWave treats each subfolder inside your catalog as one TV channel.
The folder name becomes the channel name.

Recommended layout:

  Channels/
    Channel Name/
      Shows/
        Show Title/
          S01E01 - Episode Name.mp4
          S01E02 - Episode Name.mp4
        Another Show/
      Movies/
        Cool Movie (1985).mp4
      Specials/
        Holiday Special.mp4
      Logos/
        logo.png         <-- channel bug / watermark logo

MediaWave scans for video files recursively inside each channel folder,
so nesting by category is optional. You can also put files directly
in the channel folder if you want to keep it simple.

Keeping folder names clean helps — avoid special characters that your
OS might not like (colons, slashes, that sort of thing).

SUPPORTED VIDEO FORMATS
-----------------------
  .mp4  .mkv  .avi  .mov

For best results, use MediaWave Converter to prep files before
adding them to a channel folder. Most modern MP4s work fine as-is.

CHANNEL LOGOS (OPTIONAL)
-------------------------
Drop a logo image inside a Logos/ subfolder:

  Channel Name/Logos/logo.png

Supported formats: PNG (preferred), JPG, JPEG, BMP, GIF

The logo appears as a small watermark overlay on-screen (the channel bug).
Enable it in Advanced Config.

If no logo is found, MediaWave shows a text fallback. No logo required.

CHOOSING YOUR CATALOG
---------------------
Your catalog is the folder that CONTAINS your channel folders,
not a channel folder itself.

Example: if your channels live at:
  D:\My TV\Channels\HBO Classic\
  D:\My TV\Channels\Comedy Central\

Then choose:  D:\My TV\Channels\

MediaWave remembers this path in settings. Change it any time with
"Choose Catalog..." on the main screen.

USING AN EXTERNAL DRIVE
------------------------
Your catalog can live on an external drive, flash drive, NAS, or any
folder your system can reach. External drives are welcome.

If the drive is disconnected when MediaWave starts, it will let you
know and give you a chance to reconnect or choose a different folder.

MULTIPLE CATALOGS
-----------------
You can switch between different catalog folders using "Choose Catalog..."
whenever you want. Each catalog is completely independent.
