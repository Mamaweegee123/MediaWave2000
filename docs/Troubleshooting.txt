===============================================================================
TROUBLESHOOTING
If the picture is fuzzy, the rabbit ears are probably fine.
===============================================================================

Start here if something is not working. Most problems have a quick fix.

-------------------------------------------------------------------------------
NO CHANNELS LOADED
------------------
"No playable videos were found inside: [folder]"

  - Make sure you selected the PARENT folder that contains your channel
    subfolders, not one of the channel folders itself.
  - Check that your video files end in .mp4, .mkv, .avi, or .mov.
  - Make sure the folder is readable (not on a locked or ejected drive).

"All catalog channels are disabled in My Catalog"

  - Open Advanced Config > My Catalog and enable at least one channel.

-------------------------------------------------------------------------------
CATALOG FOLDER NOT FOUND
--------------------------
"Catalog folder not found: [path]"

  - The folder was moved, renamed, or is on a disconnected drive.
  - Press "Choose Catalog..." and re-select the folder.
  - If the folder is on an external drive, plug it in first.

-------------------------------------------------------------------------------
VIDEO WON'T PLAY / BLACK SCREEN
---------------------------------
  - Try converting the file with MediaWave Converter.
  - Make sure your video drivers are up to date.
  - On Windows, check that Visual C++ Redistributable is installed
    (the installer should handle this automatically).

-------------------------------------------------------------------------------
APP OPENS TO A BLANK OR DARK SCREEN
-------------------------------------
  - This can happen if the display mode does not match your monitor.
    Try changing "Display Mode" on the main screen.

-------------------------------------------------------------------------------
LOGOS DO NOT SHOW
-----------------
  - Make sure the logo file is inside a Logos/ subfolder:
      Channel Name/Logos/logo.png
  - Make sure the channel bug (watermark) is enabled in Advanced Config.
  - Supported formats: PNG, JPG, JPEG, BMP, GIF

-------------------------------------------------------------------------------
COMMERCIALS NOT PLAYING
------------------------
  - Check that "Enable Commercials" is checked in Advanced Config.
  - Verify the Commercials folder path is set and the folder exists.
  - Make sure the folder contains .mp4 or other supported video files.
  - Reload the catalog after making changes to commercial settings.

-------------------------------------------------------------------------------
NEWTV / YOUTUBE CHANNELS NOT LOADING
--------------------------------------
NetTV and YouTube-based channels may require browser verification or
fresh cookies, depending on your setup. If a channel loads but shows
no content, try opening the source URL in a regular browser first,
then restart MediaWave. Cookie-related issues may require re-authentication
in a browser session that MediaWave can read from.

-------------------------------------------------------------------------------
CONVERTER SAYS "VIDEO TOOLS MISSING"
--------------------------------------
  ffmpeg is not installed or not on your PATH.

  macOS:   run "brew install ffmpeg" in Terminal.
  Windows: place ffmpeg.exe next to MediaWave Converter.exe,
           or install ffmpeg and add it to your system PATH.

  See Converter Guide.txt for more details.

-------------------------------------------------------------------------------
FONTS LOOK WRONG / MISSING
----------------------------
  - MediaWave bundles its own fonts, so this should be rare.
  - If a custom font is missing, check the Fonts/ folder in your app
    directory or the User Content/Fonts folder.

-------------------------------------------------------------------------------
HIGH CPU / SLOW PERFORMANCE
-----------------------------
  - Disable thumbnail caching or reduce catalog size.
  - For large libraries, let the first catalog scan finish completely
    before navigating the guide.

-------------------------------------------------------------------------------
SETTINGS RESET AFTER UPDATE
-----------------------------
  Settings are stored at:

    Windows:  %APPDATA%\MediaWave 2000\app_settings.json
    macOS:    ~/Library/Application Support/MediaWave 2000/
    Portable: <app folder>/User Content/Settings/

  Do not delete these files unless you want a full reset.

-------------------------------------------------------------------------------
STILL STUCK?
------------
  - Check the MediaWave release page for known issues and updates.
  - Report a bug with a description of what you were doing and what
    happened. Screenshots help a lot.
