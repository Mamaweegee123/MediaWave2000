============================================================
  TROUBLESHOOTING
============================================================

NO CHANNELS LOADED
------------------
Problem: "No playable videos were found inside: [folder]"

  - Make sure you selected the PARENT folder that contains
    your channel subfolders, not a channel folder itself.
  - Check that your video files end in .mp4, .mkv, .avi,
    or .mov.
  - Make sure the folder is readable (not on a locked drive).

Problem: "All catalog channels are disabled in My Catalog"

  - Open Advanced Config > My Catalog.
  - Enable at least one channel.

CATALOG FOLDER NOT FOUND
--------------------------
Problem: "Catalog folder not found: [path]"

  - The folder was moved, renamed, or is on a disconnected
    drive.
  - Press "Choose Catalog…" and re-select the folder.
  - If the folder is on an external drive, plug it in first.

VIDEO WON'T PLAY / BLACK SCREEN
---------------------------------
  - Try converting the file with MediaWave Converter.
  - Make sure your video drivers are up to date.
  - On Windows, check that Visual C++ Redistributable is
    installed (the installer should have handled this).

APP OPENS TO A BLANK OR DARK SCREEN
-------------------------------------
  - This can happen if the display mode is set to a profile
    that doesn't match your monitor. Try changing "Display
    Mode" on the main screen.

COMMERCIALS NOT PLAYING
------------------------
  - Check that "Enable Commercials" is checked in Advanced Config.
  - Verify the Commercials folder path is set correctly.
  - Make sure the folder contains .mp4 or other video files.
  - Reload the catalog after changing commercial settings.

CONVERTER SAYS "VIDEO TOOLS MISSING"
--------------------------------------
  - ffmpeg is not installed or not on your PATH.
  - Windows: place ffmpeg.exe next to MediaWave Converter.exe.
  - macOS: run "brew install ffmpeg" in Terminal.

FONTS LOOK WRONG / MISSING
----------------------------
  - The app includes its own fonts. If a custom font is
    missing, check the Fonts/ folder in your app directory
    or the User Content/Fonts folder.

HIGH CPU / SLOW PERFORMANCE
-----------------------------
  - Disable thumbnail caching or reduce catalog size.
  - For large libraries, let the first catalog scan finish
    before using the guide.

SETTINGS RESET AFTER UPDATE
-----------------------------
  - Settings are stored in:
      Windows: %APPDATA%\MediaWave2000\app_settings.json
      macOS:   ~/Library/Application Support/MediaWave2000/
      Portable: <app folder>/User Content/Settings/
  - Do not delete these files unless you want a full reset.

STILL STUCK?
------------
  - Check the MediaWave release page for known issues.
  - Report bugs with a description of what you did and
    what happened.
