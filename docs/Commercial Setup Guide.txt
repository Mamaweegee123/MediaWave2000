===============================================================================
COMMERCIAL SETUP GUIDE
Because nothing says "real TV" like an ad break.
===============================================================================

Commercials are completely optional. MediaWave runs fine without them.
If you do want that authentic mid-show interruption experience, read on.

WHAT YOU NEED
-------------
A folder of short video clips. These can be:
  - Real recorded commercials from old VHS tapes
  - Custom bumpers or promos you made
  - Station IDs, transitions, anything short and fun
  - Literally any video clips you feel like throwing in

Short clips work best — 15 to 60 seconds is the sweet spot.
Longer clips start to feel like episodes, which is a different vibe.

Recommended: run them through MediaWave Converter first for smooth playback.

SETTING UP YOUR COMMERCIALS FOLDER
------------------------------------
1. Put your commercial clips in a folder. Organize however you like —
   MediaWave scans the whole folder recursively, including subfolders
   nested inside subfolders.

   Example:
     Commercials/
       1980s/
         ad_01.mp4
         ad_02.mp4
       Bumpers/
         bumper_01.mp4

2. In MediaWave, open "Advanced Config" from the main screen.

3. Find the "Commercials" section.

4. Enable commercials and select your Commercials folder.

5. Set how often you want commercial breaks to appear.

6. Save and reload your catalog.

DEFAULT LOCATION
----------------
The User Content/Commercials/ folder included with this release is ready
to use — just drop clips in there and point Advanced Config to it.
Or choose any other folder on any drive. Whatever works.

COMMERCIAL TIMING MODES
------------------------
  Between Episodes Only      - a break plays between each episode/show
  Timed Mid-Episode Breaks   - a break plays every few minutes during
                               longer shows, like a real mid-show ad break

DENSITY SETTINGS
----------------
  Light     - 1 to 2 ads per break
  Medium    - 2 to 3 ads per break
  Heavy     - 3 to 5 ads per break

PER-CHANNEL OVERRIDES
---------------------
In Advanced Config > My Catalog, you can override commercial settings
for individual channels. Handy if one channel needs different timing.

HOW MEDIAWAVE SORTS YOUR CLIPS
--------------------------------
MediaWave reads your folder names to guess what kind of clip something is.
You don't have to name folders perfectly — this is just for organizing and
filtering your library, not a strict requirement.

  Folder name contains...           MediaWave treats clips inside as...
  ---------------------------       -------------------------------------
  "bumper" / "bumpers"               Bumpers
  "station id" / "ident(s)"          Station IDs
  "promo" / "promos" / "network      Promos
  promos"
  (anything else)                    Regular commercials

  Folder names like "80s", "90s", "2000s", "2010s", or "70s" are picked up
  as the clip's era, so you can mix and match by decade later.

  Folder names that mention things like "kids", "toys", "food", "tech",
  "games", "music", "sports", or "movies" are picked up as a rough category
  for the same reason.

BUMPERS, PROMOS & STATION IDs TODAY
--------------------------------------
Right now, bumpers/promos/station IDs are flavor mixed into your regular
commercial breaks — MediaWave occasionally slips one in alongside your ads,
based on the toggles in Advanced Config (Allow Bumpers / Allow Promos /
Allow Station IDs).

They are NOT yet a way to assign a specific bumper to a specific show
(like always playing a particular Toonami-style bumper before one show).
That per-show bumper feature is planned but not built yet.

ONE MORE THING
----------------
Commercials, bumpers, promos, and station IDs never show up as their own
entry in the Guide. The Guide always shows your actual program — ad breaks
just play in between, the same way a real TV guide doesn't list the
commercials either.

DISABLING COMMERCIALS
---------------------
Uncheck "Enable Commercials" in Advanced Config at any time.
Your commercial library stays put — nothing gets deleted.

If no commercials folder is set or the folder is empty, MediaWave
skips ad breaks silently and keeps playing. Nothing breaks.
